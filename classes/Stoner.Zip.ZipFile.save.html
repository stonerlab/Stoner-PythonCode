ZipFile.save¶

ZipFile.save(filename=None)[source]¶

Overrides the save method to allow ZipFile to be written out to disc (as a mininmalist output)

Parameters:filename (string or zipfile.ZipFile instance) – Filename to save as (using the same rules as for the load routines)
