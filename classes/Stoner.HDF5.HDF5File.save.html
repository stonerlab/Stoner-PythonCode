HDF5File.save¶

HDF5File.save(h5file=None)[source]¶

Writes the current object into an hdf5 file or group within a file in a fashion that is compatible with being loaded in again with the same class.

Parameters:h5file (string or h5py.Group) – Either a string, of h5py.File or h5py.Group object into which to save the file. If this is a string, the corresponding file is opened for writing, written to and save again.
Returns:A copy of the object
