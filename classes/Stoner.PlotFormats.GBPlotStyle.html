Stoner.PlotFormats.GBPlotStyle¶

class Stoner.PlotFormats.GBPlotStyle(**kargs)[source]¶

Template developed for Gavin’s plotting.

This is largely an experimental class for trying things out rather than for serious plotting.

Example
from Stoner import Data
from Stoner.PlotFormats import GBPlotStyle

d=Data("../sample.txt",setas="xy",template=GBPlotStyle)
d.y=d.y-(max(d.y)/2)
d.plot()
